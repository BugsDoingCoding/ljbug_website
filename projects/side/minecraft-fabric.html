Installation Instructions

  1. Install Fabric Loader for Minecraft 1.21.1 from fabricmc.net.
  2. Run the installer and select version 1.21.1.
  3. Launch Minecraft once using the new Fabric profile, then close it.
  4. Download the mod pack: Google Drive Mod Pack

Mod Folder Setup (Windows)

  1. Press Win + R, type %appdata%, and press Enter.
  2. Open .minecraft.
  3. If a mods folder exists, delete it.
  4. Move the downloaded ZIP into .minecraft.
  5. Extract it so .minecraft/mods/*.jar exists.
  6. Delete the ZIP file.

Correct structure:

.minecraft/mods/*.jar

Incorrect structures:

.minecraft/mods/mods/*.jar
.minecraft/mods (1)/mods/*.jar
