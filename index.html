How to Join

  1. Download the current recommended mods and loader installation (see notes below).
  2. Locate your mods folder in `.minecraft`, and copy the provided mods into it.
  3. Start Minecraft with the new profile and join: ip.bugsbudsmc.com

Mac Users: If the .jar file won't run, open Terminal, type open, drag the file in, then press Enter.
